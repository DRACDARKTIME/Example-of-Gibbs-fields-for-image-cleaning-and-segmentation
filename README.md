# Example-of-Gibbs-fields-for-image-cleaning-and-segmentation
This repository presents an application of Gibbs fields to image cleaning and segmentation. The idea is to take advantage of the fact that, in many images, pixel values ​​tend to vary smoothly, meaning that a pixel resembles its neighbors and abrupt changes are usually concentrated at edges or contours. Under this assumption, a Gibbs field model is constructed that assigns a higher probability to ordered images and a lower probability to configurations with abrupt and isolated changes characteristic of noise. However, since the number of possible image configurations is enormous, it is not possible to evaluate the model exactly or explore all alternatives. Therefore, Markov chain Monte Carlo simulation (MCMC) is used. The algorithm updates the image pixel by pixel, making decisions that depend on the immediate neighbor and the observed information. After a sufficient number of iterations, the method generates a collection of plausible restored images. From these realizations, a final reconstruction is built.
